# Memory-Game

**Memory-Game** is a digital version of the classic card matching game, built using **React (Vite)**, **SCSS**, and **JavaScript**.

## Dependencies

- **Sass (SCSS)** – CSS preprocessor  
- **Tailwind CSS** – Utility-first CSS framework

## Overview

The game challenges players to test their memory:

- Flip cards to reveal their values
- Match two identical cards to earn points
- Find all matching pairs to win the game

## Features

- Interactive digital card grid
- Scoring system based on matched pairs
- Simple, intuitive gameplay

## Running Locally
To run the game locally:

1. Download files
2. Get in to directory by terminal
3. Install dependencies
```bash
npm i
```
4. Use in terminal
```bash
npm run dev
```
